The float property has four possible values: left | right | inherit | none

Floating it to move to the left-most or right-most edge of its parent container. Floating removes the element from normal flow, and will cause elements below it to shift accordingly.

aside, article, section, header, footer, nav { display: block; } html, body { margin: 0; padding: 0; } html { background: #ccc; } body { width: 600px #fff; margin: 2em auto 2em; } div { margin-bottom: 1em; width: 100px; height: 100px; } /* Custom element styles */ .element1 { background: rgb(211, 206, 61); float: left; margin-right: 1em; } .element2 { background: rgb(85, 66, 54); float: left; margin-right: 1em; } .element3 { background: rgb(247, 120, 37); float: left; margin-right: 1em; }
